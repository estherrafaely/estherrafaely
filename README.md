# 👩🏻‍💻 Esther Rafaely

**`Desenvolvimento de Software • Aprendizado Contínuo`**

Oi, me chamo Esther, e venho me apaixonado cada vez mais pela área da tecnologia. Comecei no curso Técnico em Informática pelo SENAI e, desde então, venho buscando aprender, evoluir e transformar conhecimento em prática. Atualmente curso Análise e Desenvolvimento de Sistemas pela UEPB e realizo estágio no Hospital Universitário da UFCG. Meu foco está no desenvolvimento de software, enquanto fortaleço minha base em programação, Programação Orientada a Objetos, Estruturas de Dados e Banco de Dados.

Acredito que sempre há algo novo para aprender e estou construindo minha trajetória um projeto de cada vez, buscando criar soluções que façam a diferença.
